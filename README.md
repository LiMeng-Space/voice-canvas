# Voice Canvas - AI 语音绘图工具

一个纯语音控制的浏览器绘图工具。用户通过普通话指令完成画布创作，应用会把语音识别结果拆解成绘图动作并立即执行。

## 作品定位

- 开发窗口：2026-06-12 00:00 至 2026-06-14 23:59
- 目标：展示语音指令理解、复杂指令拆解、低延迟绘图反馈和容错提示能力

## 作品亮点

- 纯语音控制：授权麦克风后，绘图、撤销、重做、清空和导出都可通过语音完成。
- 复杂指令拆解：支持把“然后、再、同时、和”等连接词拆成多个绘图动作。
- 口语容错：支持“生成、创建、放置、写上、底色、顶部、底部、正中”等自然表达。
- 语音识别自动修正：按动作、颜色、图形、方位、服装鞋帽等类别修正常见同音/近音误识别，例如“眼视→演示”“戴眼睛→戴眼镜”“羽毛求→羽毛球”“身分→山峰”“下发→下方”。
- 快速演示：说“播放演示”可自动生成完整场景，降低 Demo 视频录制风险。
- 运动场景模板：说“我喜欢打羽毛球”可生成球场、球网、人物、球拍和羽毛球组合画面。
- 局部修改与删除：支持把最近人物的上衣、下装、鞋子或帽子改成新颜色/类型，也支持删除最近一个元素。
- 目标选择：支持按位置或类型修改/删除，例如“把左边的人上衣改成白色”“删除右上角的太阳”。
- 执行反馈：侧栏展示绘图步数、撤销/重做状态、最近结果和执行记录。
- 本地实现：无后端、无第三方框架、无 API 密钥，便于评委直接复现。

## 本地运行

本项目不依赖第三方包，推荐使用 Chrome 或 Edge 打开。

```bash
python -m http.server 5173
```

然后访问：

```text
http://localhost:5173
```

首次使用时，浏览器会要求麦克风权限。受浏览器安全策略限制，可能需要点击一次“启动麦克风”；之后绘图、撤销、清空、导出等操作均通过语音完成。

## 已支持指令

- 绘制形状：圆形、正方形、矩形、三角形、线条、星星、爱心、云朵、太阳、树、房子、花朵、山峰
- 绘制人物：支持性别/年龄段、发型、眼镜、高矮、体型、肤色、上衣、下装、鞋子类型、帽子类型等组合属性
- 修改人物：例如“蓝色上衣修改为白色上衣”“把左边的人上衣改成白色”“裤子改成黑色短裤”“鞋子改成白色运动鞋”
- 设置颜色：红色、橙色、黄色、绿色、青色、蓝色、紫色、粉色、棕色、灰色、黑色、白色、浅蓝色、浅绿色、米色
- 设置位置：左上、右上、左下、右下、中间、上方、下方、左边、右边
- 设置数量：一到十，或直接说数字
- 设置样式：填充、实心、描边、空心、粗线、细线
- 画布操作：撤销、重做、清空画布、删除最近一个元素、删除指定类型或位置元素、导出作品
- 文字操作：例如“写文字 夏日实训 在中间”
- 演示模式：例如“播放演示”，自动生成一幅完整场景，方便 demo 视频展示
- 场景模板：例如“我喜欢打羽毛球”“画我在球场打球”“播放羽毛球演示”，生成羽毛球运动场景
- 执行反馈：展示绘图步数、撤销/重做状态、最近一次执行结果和完整执行记录
- 复杂指令拆解：例如“画一个红色太阳在左上角，然后画三朵蓝色云在上方，再写文字 夏日实训 在中间”
- 容错表达：支持“生成、创建、放置、写上、底色、顶部、底部、正中、十一朵”等更口语化说法，并修正常见语音识别同音字

## 依赖说明

- Web Speech API：浏览器内置语音识别能力
- Canvas 2D API：浏览器内置绘图能力
- Web Speech Synthesis API：浏览器内置语音反馈能力

没有使用第三方绘图框架、语音服务或云端大模型。指令解析、复杂指令拆解和绘图执行逻辑均在 `app.js` 中实现。

## 安全说明

本项目没有后端服务，也不需要 API key、账号密码或云服务密钥。仓库中不应提交任何私密凭据。
