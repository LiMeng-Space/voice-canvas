# Voice Canvas - AI 语音绘图工具

一个纯语音控制的浏览器绘图工具。用户通过普通话指令完成画布创作，应用会把语音识别结果拆解成绘图动作并立即执行。

## 作品定位

- 议题：AI 语音绘图工具
- 开发窗口：2026-06-12 00:00 至 2026-06-14 23:59
- 目标：展示语音指令理解、复杂指令拆解、低延迟绘图反馈和容错提示能力

## 本地运行

本项目不依赖第三方包，推荐使用 Chrome 或 Edge 打开。

```bash
python -m http.server 5173
```

然后访问：

```text
http://localhost:5173
```

首次使用时，浏览器会要求麦克风权限。受浏览器安全策略限制，可能需要点击一次“启动麦克风”；之后绘图、撤销、清空、导出等操作均通过语音完成。

## 已支持指令

- 绘制形状：圆形、正方形、矩形、三角形、线条、星星、爱心、云朵、太阳、树、房子、花朵、山峰
- 设置颜色：红色、橙色、黄色、绿色、青色、蓝色、紫色、粉色、棕色、灰色、黑色、白色、浅蓝色、浅绿色、米色
- 设置位置：左上、右上、左下、右下、中间、上方、下方、左边、右边
- 设置数量：一到十，或直接说数字
- 设置样式：填充、实心、描边、空心、粗线、细线
- 画布操作：撤销、重做、清空画布、导出作品
- 文字操作：例如“写文字 夏日实训 在中间”
- 演示模式：例如“播放演示”，自动生成一幅完整场景，方便 demo 视频展示
- 执行反馈：展示绘图步数、撤销/重做状态、最近一次执行结果和完整执行记录
- 复杂指令拆解：例如“画一个红色太阳在左上角，然后画三朵蓝色云在上方，再写文字 夏日实训 在中间”
- 容错表达：支持“生成、创建、放置、写上、底色、顶部、底部、正中、十一朵”等更口语化说法

## 示例指令

```text
把背景改成浅蓝色
画一个黄色太阳在左上角
再画三朵白色云在上方
画两座绿色山峰在下方
写文字 夏日实训 在中间
撤销
重做
导出作品
底色换成米色
生成十二颗黄色星星在顶部
写上 Voice Canvas 在正中
播放演示
```

## 依赖说明

- Web Speech API：浏览器内置语音识别能力
- Canvas 2D API：浏览器内置绘图能力
- Web Speech Synthesis API：浏览器内置语音反馈能力

没有使用第三方绘图框架、语音服务或云端大模型。指令解析、复杂指令拆解和绘图执行逻辑均在 `app.js` 中实现。
