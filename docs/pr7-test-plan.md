# PR7 测试计划：羽毛球运动场景模板

## 测试目标

验证语音绘图工具可以识别更自然的场景化表达，并自动生成包含球场、球网、人物、球拍和羽毛球的组合画面。

## 测试指令

```text
我喜欢打羽毛球
画我在球场打球
画一个戴眼镜的男生在球场打羽毛球
播放羽毛球演示
撤销
重做
```

## 预期结果

- “我喜欢打羽毛球”可以被识别为场景指令，而不是提示未理解。
- 画布会生成浅绿色羽毛球场、白色场地线、球网、两个人物、两个球拍和一个羽毛球。
- “画我在球场打球”在没有明确指定运动项目时，默认生成羽毛球运动场景。
- “画一个戴眼镜的男生在球场打羽毛球”会把人物属性带入主角。
- “播放羽毛球演示”会进入羽毛球场景，而不是普通演示模式。
- 执行日志会记录“生成羽毛球场景”。
- 撤销后能回到场景生成前的画布，重做后能恢复羽毛球场景。
