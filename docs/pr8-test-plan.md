# PR8 测试计划：最近元素修改与删除

## 测试目标

验证语音绘图工具可以在已绘制内容基础上继续修改最近人物的服装/鞋帽属性，并支持删除最近一个绘图元素。

## 测试指令

```text
画一个戴眼镜的短发男生在中间
蓝色上衣修改为白色上衣
裤子改成黑色短裤
鞋子改成白色运动鞋
写文字 测试 在中间
删除最近一个元素
撤销
重做
```

## 预期结果

- 绘制人物后，系统会记录该人物为可编辑元素。
- “蓝色上衣修改为白色上衣”会把最近人物的上衣改成白色。
- “裤子改成黑色短裤”会把最近人物的下装改成黑色短裤。
- “鞋子改成白色运动鞋”会把最近人物的鞋子改成白色运动鞋。
- “删除最近一个元素”会删除最后绘制的文字或图形元素。
- 修改或删除后，画布会基于已有内容重新渲染，保留其他元素。
- 撤销后能回到上一步，重做后能恢复修改或删除结果。
- 执行日志会记录修改、删除、撤销和重做结果。

## 当前边界

- PR8 只支持修改最近一个人物，不支持指定“左边的人”或“第二个人”。
- PR8 的删除能力只删除最近一个元素，不支持按类型或位置删除；这些能力计划放到 PR9。
