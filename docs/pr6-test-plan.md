# PR6 测试计划：可配置人物绘制

## 测试目标

验证语音绘图工具可以绘制带组合属性的人物，而不是只支持固定的男孩、女孩或老人图标。

## 测试指令

```text
画一个戴眼镜的短发男生在中间
画一个高个子深肤色长发女生在右边
画一个微胖的老人在下方
画一个矮个子浅肤色卷发小孩在左边
画两个蓝衣服戴眼镜的人在上方
画一个戴帽子穿红色卫衣黑色长裤白鞋的女生在中间
画一个穿蓝色T恤绿色短裤黑鞋的小孩在左边
画一个穿紫色衬衫灰色长裙棕色鞋子的长发女人在右边
画一个戴棒球帽穿红色外套黑色长裤白色运动鞋的男生在中间
画一个戴渔夫帽穿蓝色T恤绿色短裤黑色凉鞋的小孩在左边
画一个戴毛线帽穿黄色毛衣灰色长裤棕色皮鞋的老人
画一个戴草帽穿紫色衬衫灰色长裙棕色靴子的长发女人在右边
画一个戴头盔穿绿色外套黑色短裤白色运动鞋的人在左边
```

## 预期结果

- 人物会显示头部、身体、手臂和腿部。
- “戴眼镜”会画出眼镜。
- “短发、长发、卷发、马尾、光头”会影响头发形状。
- “高个子、矮个子”会影响人物高度比例。
- “瘦、微胖、强壮”会影响身体宽度。
- “浅肤色、中等肤色、深肤色”会影响头部和手臂颜色。
- “红衣服、蓝衣服、绿色上衣”等颜色表达会影响上衣颜色。
- “黑色长裤、绿色短裤、灰色长裙”等表达会影响下装类型和颜色。
- “运动鞋、皮鞋、靴子、凉鞋”等表达会影响鞋子类型。
- “白鞋、黑鞋、棕色鞋子”等表达会影响鞋子颜色。
- “棒球帽、鸭舌帽、渔夫帽、毛线帽、草帽、宽檐帽、头盔”等表达会影响帽子类型。
- “戴帽子、红帽子”等表达会显示帽子并影响帽子颜色。
- 执行日志会记录绘制的人物描述。
