# Demo 视频脚本

## 建议时长

2 到 4 分钟。

## 录制流程

1. 打开 `http://localhost:5173`，允许麦克风权限。
2. 介绍作品：这是一个纯语音控制的 AI 语音绘图工具，绘图、撤销、重做、清空和导出都通过语音完成。
3. 说第一条指令：

```text
把背景改成浅蓝色
```

4. 说复杂指令：

```text
画一个黄色太阳在左上角，然后画三朵白色云在上方，再画两座绿色山峰在下方
```

5. 说文字指令：

```text
写文字 夏日实训 在中间
```

6. 展示编辑能力：

```text
撤销
重做
```

7. 展示导出能力：

```text
导出作品
```

8. 简短说明技术实现：

```text
系统使用浏览器 Web Speech API 做语音识别，先把语音转成文本，再通过本地规则解析拆分复杂指令，最后用 Canvas 2D API 执行绘图。当前版本没有接入云端模型，因此部署简单，也不会上传语音内容。
```

## 视频检查清单

- 画面能看清浏览器地址和画布。
- 声音能听清每条语音指令。
- 至少展示一次复杂指令拆解。
- 至少展示一次撤销或重做。
- README 中填入可访问的视频链接。
