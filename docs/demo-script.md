# Demo 视频脚本

## 建议时长

2 到 4 分钟。

## 录制流程

1. 打开 `http://localhost:5173`，允许麦克风权限。
2. 介绍作品：这是一个纯语音控制的 AI 语音绘图工具，绘图、撤销、重做、清空和导出都通过语音完成。
3. 说第一条指令：

```text
把背景改成浅蓝色
```

4. 说复杂指令：

```text
画一个黄色太阳在左上角，然后画三朵白色云在上方，再画两座绿色山峰在下方
```

5. 展示演示模式：

```text
播放演示
```

6. 展示运动场景模板：

```text
我喜欢打羽毛球
```

可以补充说明：这条口语化指令会直接生成羽毛球场景，包含球场、球网、人物、球拍和羽毛球；如果说“画一个戴眼镜的男生在球场打羽毛球”，人物特征也会被带入主角。

7. 展示局部修改能力：

```text
画一个戴眼镜的短发男生在中间
蓝色上衣修改为白色上衣
鞋子改成白色运动鞋
```

可以补充说明：系统会记录最近绘制的人物元素，因此可以在画完后继续修改人物的上衣、下装、鞋子或帽子。

8. 展示目标选择能力：

```text
画一个蓝色短发男生在左边
画一个红色长发女生在右边
把左边的人上衣改成白色
画一个黄色太阳在右上角
删除右上角的太阳
```

可以补充说明：系统不只支持最近元素，也可以根据位置或元素类型选择要修改/删除的目标。

9. 说文字指令：

```text
写文字 夏日实训 在中间
```

10. 展示删除和撤销能力：

```text
删除最近一个元素
撤销
重做
```

11. 展示导出能力：

```text
导出作品
```

12. 简短说明技术实现：

```text
系统使用浏览器 Web Speech API 做语音识别，先把语音转成文本，再通过本地规则解析拆分复杂指令，最后用 Canvas 2D API 执行绘图。当前版本没有接入云端模型，因此部署简单，也不会上传语音内容。
```

## 视频检查清单

- 画面能看清浏览器地址和画布。
- 声音能听清每条语音指令。
- 至少展示一次复杂指令拆解。
- 至少展示一次撤销或重做。
- README 中填入可访问的视频链接。
