# PR9 测试计划：按类型和位置选择元素

## 测试目标

验证语音绘图工具可以在 PR8 的最近元素基础上，进一步按元素类型或画面位置选择要修改/删除的目标。

## 测试指令

```text
画一个蓝色短发男生在左边
画一个红色长发女生在右边
把左边的人上衣改成白色
画一个黄色太阳在右上角
删除右上角的太阳
播放眼视
画一个戴眼睛的短发男生在中间
我喜欢打羽毛求
画两座绿色的身分在下发
化一个黄涩泰阳在左上脚
画三朵白色云多在上访
画一个带眼睛穿红色味衣黑色长库白学的人在中见
写文字 测试 在中间
删除文字
删除右边的人
撤销
重做
```

## 预期结果

- “把左边的人上衣改成白色”会选择左边的人物，而不是最近绘制的右边人物。
- “删除右上角的太阳”会按类型和位置删除右上角太阳。
- “删除文字”会删除最近一个文字元素。
- “删除右边的人”会按位置删除右边人物。
- “播放眼视”会被自动修正为“播放演示”，并生成自动演示场景。
- “戴眼睛”会被自动修正为“戴眼镜”，人物会显示眼镜。
- “羽毛求”会被自动修正为“羽毛球”，并生成羽毛球运动场景。
- “身分”和“下发”会被自动修正为“山峰”和“下方”，并绘制绿色山峰。
- “黄涩泰阳”“云多”“左上脚”“上访”等会被自动修正为对应的颜色、图形和方位词。
- “带眼睛、味衣、长库、白学、中见”等会被自动修正为人物和服装相关标准词。
- 修改或删除后，画布会基于底图和剩余元素重新渲染，保留未被选中的元素。
- 撤销后能回到上一步，重做后能恢复目标修改或目标删除结果。
- 执行日志会记录左边、右边、右上方等目标范围。

## 当前边界

- PR9 支持按位置和元素类型选择目标，不支持“第二个太阳”这类编号选择。
- PR9 仍以本地规则解析为主，不引入云端大模型或第三方依赖。
