# PR2 测试计划：语音指令容错增强

## 测试目标

验证应用能理解更自然的中文口语表达，而不是只依赖固定模板。

## 测试指令

```text
底色换成米色
生成十二颗黄色星星在顶部
创建一个红色爱心在正中
放置两棵绿色小树在底部
写上 Voice Canvas 在正中
三朵白色云在上方
坐标200,160放一个蓝色圆形
```

## 预期结果

- “底色”会被识别为背景操作。
- “生成、创建、放置”会被识别为绘图意图。
- “顶部、底部、正中”会映射到正确位置。
- “十二颗、两棵、三朵”会解析为数量，数量上限限制为 12。
- 没有说“画”的“三朵白色云在上方”也能执行绘图。
- 坐标表达能把图形放到指定位置附近。
